# Last Invocation

Type: event

## Overview

The Crown Eidolon ( 究極召喚獣, Kyūkyoku Shōkanjū?, lit. Final Summoned Monster ) is the eidolon an invoker gains at the end of their pilgrimage to Asterreach. The eidolon is intended to be used only once: during the final battle with The Hollow Eclipse, in a ritual called the Last Invocation ( 究極召喚, Kyūkyoku Shōkan?, lit. Last Invocation ).

## Details

Recovered notes about Last Invocation remain fragmentary, yet the surviving records still connect it to Elyndran institutions, pilgrimage-era practices, and a broader network of related entities inside the synthetic corpus.
