# Isola Veyn

Type: location

## Overview

A large beach that contains a port. It is the only way on and off Isola Veyn.

## Details

A large lake in the center of Isola Veyn. It contains a long underwater passage that can be used to reach the village.

Veyn Temple is located in Veyn Village. It is where the eidolon, Aurelion, is obtained from the temple's dreambound, a young girl. It is where Elyra Noctis completes her studies and becomes an invoker.
